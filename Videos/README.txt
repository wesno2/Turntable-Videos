Each .pds is a PowerDirector project.  The projects refer to image files found in the
Turntable-Videos/Pictures directory.

Open each .pds with PowerDirector.  PowerDirector will complain that it cannot locate the
referenced image files.  Instruct PowerDirector to look for the images on the local directory
into which you've stored the contents of the Turntable-Videos/Picture directory.

From within PowerDirectory, you produce 3 mp4 files as shown below.

e03.ccw.pds is used to produce:
e03.ccw.h.mp4
e03.ccw.n.mp4
e03.ccw.t.mp4

e03.ccw.pds is used to produce:
e03.cww.h.mp4
e03.cww.n.mp4
e03.cww.t.mp4

e95.ccw.pds is used to produce:
e95.ccw.h.mp4
e95.ccw.n.mp4
e95.ccw.t.mp4

e95.cww.pds is used to produce:
e95.cww.h.mp4
e95.cww.n.mp4
e95.cww.t.mp4

After the .mp4 files are created, open the Nextion Editor.
Within the Nextion Editor, create 1 .video file for each .mp4 file.
See the "Control Panels" video for the settings to use to create
the .video files.

Because the generated .mp4 and .video files are larger than 25mb,
they cannot be stored on github.

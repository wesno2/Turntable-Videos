# Turntable-Videos
Auxiliary files created to support viewers of the turntable videos on YouTube if they wish to build their own turntable.
Examples include:
1) files which contain detailed information that would be too tedious to cover during the videos
2) the Arduino program used to control the turntable
3) the Nextion .HMI file used to control the Nextion
4) the image files that are uploaded to the Nextion and displayed on command

Folder: Circuitry

Files referenced in the Circuitry video.
Files:  
Wiring Plan for a Prototype PCB.pdf
  1 page diagram showing how you would connect the external components to each other and to the Arduino using a prototype PCB

Folder: Nextion Display
  Files that are uploaded to the Nextion
  Files:
    something.HMI
      the main control program uploaded to define the panels and how they are programmed
  Folder:  Pictures
    All of the "picture" files that are uploaded to the Nextion and referenced by the .HMI file.  Also contains an index to all of them.
    pictureindex.pdf
      index to all of the picture files
    *.jpg or *.png
      the picture files that are uploaded
  Folder: source files for pictures
    Most of the picture files are custom created using Photoshop.  The source files used to create the picture files are stored here.  Also contains an index to all of them.
      sourceindex.pdf
      *.psd

Folder: Arduino Program
  Program that is uploaded to the Arduino.
  Files:
    something.ino
      the main control program

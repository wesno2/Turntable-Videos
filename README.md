# Turntable-Videos
Auxiliary files created to support viewers of the turntable videos on YouTube if they wish to build their own turntable.

Folder: Circuitry

   Files referenced in the Circuitry video.

Folder: Nextion Display
Files that are uploaded to the Nextion

Folder:  Pictures

All of the "picture" files that are uploaded to the Nextion, along with their source files.

Folder: Videos

All video files that were uploaded to the Nextion, along with their source files.
  
Folder: Program
The Arduino program and its support files.

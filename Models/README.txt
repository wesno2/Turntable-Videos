This directory contains the Sketchup models which show the turntable unit's construction.  If you wish to view the models using a 3D viewer, see the instructions below.

VIEWING SKETCHUP MODELS
To view, in 3D, one of the models listed below:
1)	click on the link to first see the model in static form
2)	click the *3D Preview* button
3)	hover over the image and when the *View in 3D* button appears, click it.
The 3D image viewer loads in the window.  
Controls are located in a palette. Click on a control to switch to using it.
To view the model in full screen, click the faint 4-arrow icon in the lower right.

PRIMARY TOOLS USED WHEN VIEWING SKETCHUP MODELS
Using the *orbit* tool (a line with a revolving arrow through it):
Click and hold the left mouse button and move it around to see the model rotate left and right and up and down.  
To quickly pan without rotating, hold down the Shift key.  
To zoom in or out, using the mouse wheel.
